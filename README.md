# ⏱️ Python Countdown Timer

## 📌 Description

A simple countdown timer built in Python that counts down from a given number of seconds.

## 🚀 Features

* Countdown timer
* Displays time in HH:MM:SS format
* Real-time delay using time module

## 🛠️ Tech Used

* Python

## ▶️ How to Run

1. Run the file:

   ```bash
   python countdown_timer.py
   ```

## 💡 Example

Enter time in seconds: 10

Output:
00:00:10
00:00:09
...
